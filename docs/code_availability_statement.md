# Suggested code availability statement

The source code, documentation, and a reproducible synthetic-data demonstration of PSIN are available at:

```text
https://github.com/<YOUR_USERNAME>/<YOUR_REPOSITORY_NAME>
```

The repository includes the PSIN model implementation, Toeplitz-Ricker initialization, self-supervised training scripts, operator-evolution analysis, installation instructions, and a synthetic dataset for testing. Some field datasets used in the manuscript are not redistributed due to data-use restrictions; however, the repository provides a synthetic dataset and scripts that allow reviewers and readers to test the complete computational workflow and apply the method to local seismic sections.
